# 🔒 CP-LOCKOUT

A competitive programming 1v1 lockout platform featuring automated Codeforces handle ownership verification and real-time match tracking.

🚀 **Live App:** [https://cp-lockout-two.vercel.app](https://cp-lockout-two.vercel.app)

---

## ✨ Features

* **Automated Codeforces Verification:** Verifies handle ownership via temporary verification tokens rendered on the user's Codeforces profile.
* **Full-Stack Authentication:** Integrated Supabase Auth with Node.js/Express backend validation.
* **1v1 Lockout Match Engine:** Real-time problem tracking using Codeforces REST APIs.

---

## 🛠️ Tech Stack

* **Frontend:** React, Vite, Tailwind CSS
* **Backend:** Node.js, Express.js, Axios
* **Database & Auth:** Supabase (PostgreSQL, Service Role Keys)
* **Deployment:** Vercel (Frontend), Render (Backend)
